## แนวคิด

- ฟ้า | ดิน (เทียน ตี๊)
    > _ความเชื่อเกี่ยวกับท้องฟ้า และพื้นดิน ซึ่งแสดงถึงทิวทัศน์ ความงามของธรรมชาติ_

    <img src="https://i.pinimg.com/originals/87/9f/75/879f75ba0e56f4179f04de8533ceb2c8.jpg" width="500" height="300" />

    - #### ภาพทิวทัศน์ 
        > _ถือว่าเป็น `จิตรกรรมประจำชาติ`_
        - รูปแบบ
            + แนวตั้ง : แขวน
                - **มองบนลงล่าง** `ฟ้า -> ดิน`
            + แนวนอน : คลี่
                - **มองซ้ายไปขวา**
            > _เมื่อชมเสร็จแล้วจะม้วนเก็บ ไม่แขวนโชว์_
            >  - _มีไว้สำหรับเป็นของขวัญ ไม่ได้ไว้ขาย_
        - #### องค์ประกอบภาพจิตรกรรม
            + [ภาพวาด](#ภาพวาด)
            + [ตราประทับ](#ตราประทับ)
            + [อักษรวิจิตร](#อักษรวิจิตรจีน)
            

            #### ภาพวาด
            - บัญญัติ 6 ประการจิตรกรรมจีน
                - `ส่งผ่านด้วยการคัดลอก`
                    > _ศึกษาจากรุ่นก่อน -> เป็นการเคารพ **การยกย่อง**_
                - ภาพมีจิตวิญญาณ
                - วิธีใช่พู่กัน
                - สื่อสารวัตถุ
                - แบบอย่างเหมาะสม
                - แบ่งและจัดวาง
            
            #### ตราประทับ
            - `เทียบเท่าลายเซน`
                > _ผู้ที่ครองภาพต่อแล้วประทับแสดงถึงความ ประทับใจ_
                - มักประทับด้านล่าง
                - แนวนอนจะไว้ฝั่งซ้าย

            <img src="https://farm2.staticflickr.com/1722/42786789312_e52708fe87_b.jpg" width="500" height="250" />

            #### อักษรวิจิตรจีน
            > _ถ่ายทอดด้วยพู่กันและหมึก [Calligraphy]_
             - `ความงามเทียบเท่างานศิลปะ`
                + ปรัชญาในตัวอักษร

## วัดพุทธ

- #### ซุ้มประตูทางเข้า (牌坊)

    <img src="https://upload.wikimedia.org/wikipedia/commons/b/bd/%E9%BB%83%E6%B0%8F02.jpg" width="500" height="300" />

    + ไผฝาง 
        > _มีลักษณะเรียบง่าย สีไม่ฉูดฉาด_
    + ไผโหลว 
        > _ตกแต่งด้วยกระเบื่อง สีสันสดใส_
        
- #### อาคารทางเข้าหน้าวัด (อาคารซานเหมิน)

    <img src="https://readthecloud.co/wp-content/uploads/2019/02/travel-wat-pumenbaoensi-2.jpg" width="500" height="300" />


    + ทางเข้าหันหน้าเข้าทิศใต้
        > _ทิศเหนือคือทิศดี ทำให้ต้องทำให้หัวชี้ไปทางทิศเหนือ ด้านหน้าของ วัด | วัง จึงหันหน้าเข้าทิศใต้_
    + หลังจากนั้นจะเป็นทางเดินยาวเข้าไป

- #### สิงโต
    > _แสดงถึงอำนาจ โดยมีสิงโตตัวผู้และตัวเมีย_
    + ตัวผู้ -> เลี้ยงบอล
    + ตัวเมีย -> เลี้ยงลูก

- #### เทพประตู [ทวารบาล] (เหมินเสิน)
    > _เทพคุ้มครองประตู_

    <img src="https://www.thairath.co.th/media/NjpUs24nCQKx5e1ExVMPRNf59weHU6ZqapBpdSs7tHm.jpg" width="500" height="300" />

    + หน้าดำ (อวี้ฉือจิ้งเต๋อ)
    + หน้าแดง (ฉินซูเป่า)
        > _มีการวาดทั้งด้านหน้าและหลังประตู_

- #### วิหารเทพ (เทียนหวางเตี้ยน)
    + `พระเมตไตรยพระพุทธเจ้า` (พระพุทธเจ้าในอนาคต) (พระอ้วน)
        + มีเทพคุ้มกันอยู่ 4 องค์ ซ้าย 2, ขวา 2
            - เหนือ (ท้าวกุเวร)
            - ใต้ (ท้าววิรุฬหก)
            - ออก (ท้าวธตรฐ)
            - ตก (ท้าววิรูปักษ์)
    + `พระเวทโพธิสัตว์` 
        > _หลังชนพระเมตไตร | หันหน้าเข้าอุโบสถ_

- #### พระอุโบสถ (ต้าสงเป่าเตี้ยน)
    > _ต้าสง = พระศรีศากยมุนี (พระพุทธเจ้า) -> นักบวชแห่งศากยะ_ 
    + `พระประธาน` (ซานฝอ)
        > _แบ่งออกเป็น 3 รูปแบบแล้วแต่ว่าแต่ละวัดจะใช้วิธีแบบไหน_
        > - _มี 3 องค์ คือ ซ้าย - กลาง - ขวา_ 
        > - _มักมี `สวัสดิกะ` ตรงอก_

        <img src="https://previews.123rf.com/images/kobby_dagan/kobby_dagan0804/kobby_dagan080400147/2852918-interior-of-chinese-buddhist-shrine-in-the-city-of-shanghai-china.jpg" width="500" height="300" />


        - [ตามเวลา](#ตรีกาล) : **ตรีกาล** = เวลาเป็นเกณฑ์
        - [ตามทิศ](#ตรีทิศ) : **ตรีทิศ** = ทิศเป็นเกณฑ์
        - [ตามกาย](#ตรีกาย) : **ตรีกาย** = กายเป็นเกณฑ์

            ### ตรีกาล
            - **พุทธเจ้าในอดีต** (_พระทีปังกรพุทธเจ้า_)
            - **พุทธเจ้าในปัจจุบัน** (_พระศากยมุนีพุทธเจ้า_)
            - **พุทธเจ้าในอนาคต** (_พระเมตไตรยพุทธเจ้า_)

            ### ตรีทิศ
            - **ความรู้** (_พระอมิตาภะพุทธเจ้า_)
            - **พระพุทธเจ้า** (_พระศากยมุนีพุทธเจ้า_)
            - **การรักษา** (_พระไภษัชยคุรุพุทธเจ้า_)

            ### ตรีกาย
            - **กายในอดีต** (สัมโภคกาย)
            - **กายที่มองไม่เห็น** (ธรรมกาย)
            - **กายที่เห็นได้ด้วยตา** (นิมรกาย)

    - `พระอานนทร์` + `พระมหากัสสปะ` 
        > _คติมหายาน : พระอานนท์อยู่เคียงข้างพระพุทธเจ้าตลอดเวลา | พระมหากัสสปะมาทันพระพุทธเจ้าตาย_

        - พระประธานงานศพ
            + พระมหาสถามปราปต์โพธิสัตว์
            + พระพุทธเจ้า
            + พระโพธิสัตว์อวโลกิดตศวรกวนอิม (_เจ้าแม่กวนอิม_)
    
    - อรหันต์ 18 องค์ 
        > _อยู่รอบๆ พระอุโบสถ_

## ฉาน
_นิกายย่อยของมหายาน ในญี่ปุ่นเพี้ยนไปเป็น " เซน "_  
> _สายของพระมหากัสสปะ คือ การถ่ายทอดจิตสู่จิต_
- #### คำสอน
    - การเพ้งจิต สู่จิตดั้งเดิม  ควบคุมอารมณ์จนมีสมาธิ
    - รู้ตระหนักในชีวิต ตระหนักรู้ด้วยตนเอง
        
- พระโพธิธรรม (ตั๊กม้อ)
    > _พระจากอินเดียที่เข้ามาจีนและนำโยคะมาฝึกกำลัง แต่คนจีนไม่รู้จักทำให้เรียกว่า "เส้าหลิน" (กำลังภายใน)
    > + ญี่ปุ่นเรียกว่า "ดารุโมะ"

- #### เหลียงข่าย
    <img src="https://ae01.alicdn.com/kf/HTB1nr8oIXXXXXbHXFXXq6xXFXXX9/Oriental-Asian-art-Original-Traditional-Chinese-painting-landscape-painting-handpainted-famous-Artist-Chinese-ink-brush-painting.jpg_640x640.jpg" width="500" height="250" />

    > นั่งสมาธิเสร็จแล้วจึงวาดหมึกด้วยวิธีการปาด จนเกิดเป็นภาพ
    - การวาดหมึก เซียน

## เต๋า
> ในญี่ปุ่นเรียกว่า โด แปลว่ากุญแจ
- #### แนวคิด
    - ไม่ขัดแย้งกับธรรมชาติ เป็นหนึ่งเดียวกับธรรมชาติ
        + `เล่าจื้อ` : ศาสดา
        > "เต็าที่ไม่เรียกขานได้ ไม่ใช่เต๋า เต๋าที่เรียกขานไม่ได้ ไม่ใช่นามแท้"

- #### สัญลักษณ์
    - **หยินยาง**
        > ในทุกสิ่งจะอยู่เป็นคู่ ดำมีขาว บวกมีลบ
        + มืด = หยิน
        + ขาว = หยาง
        + จุดที่อยู่ในขาวดำ : ไม่มีอะไรดำสุดหรือขาวสุด คนชั่วก็ยังมีส่วนที่ดีอยู่     
    
    - แผนภูมิ 8 ทิศ (ปา กว้า)
        + เข็มขาด = หยิน
        + เข็มเต็ม = หยาง 
        + **ธาตุทั้ง 5** (ฟงซุ่ย)
            - ดิน
            - น้ำ
            - ลม
            - ไฟ
            - โลหะ
        
        + **หวงจุ้ย**
            - การวางชัยภูมิที่ดี : มีเหตุผลรองรับ
        
## ชุดฮ่องเต้
- #### ความหมายของสัญลักษณ์
    + **มังกร**
        - สัญลักษณ์แห่งจักรพรรดิ์
            - กำแพงเก้ามังกร : แบบอย่างมังกรที่สวยที่สุด
            - พระราชวังต้องห้าม : มังกรด้านเมฆ
            > - จักรพรรดิใช้มังกร 5 เล็บ
            > - ราชวงศ์ใช้ 4 เล็บ
            > - ขุนนางใช้ 3 เล็บ
        
        - **เสื้อคลุมมังกร**
            - สัญลักษณ์ทั้ง 12 ประการ
                - มังกร
                - กลุ่มดาว
                - ดวงอาทิตย์
                - ไก่ฟ้า
                - หมู่เขา
                - ดวงจันทร์
                - เมล็ดข้าว
                - สาหร่าย
                - จอกบูชา
                - เปลวไฟ
                - ขวาน
                - อักษรฝู
